# AI-Agent Frontend

Frontend MVP para operar el agente de WhatsApp multi-tenant.

## Stack

- React + Vite
- TypeScript estricto
- Arquitectura hexagonal
- Tailwind CSS
- TanStack Query
- Vitest + Testing Library + MSW

## Variables de entorno

Copia `.env.example` a `.env` y ajusta:

```bash
VITE_API_BASE_URL=http://localhost:8000
```

## Comandos

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm run test
npm run build
```
